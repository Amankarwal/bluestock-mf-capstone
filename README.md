# Mutual Fund Analytics Capstone Project

Data Analyst Internship Project at Bluestock Fintech

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Project Tasks
- Data Ingestion
- Data Quality Checks
- Exploratory Data Analysis (EDA)
- Fund Performance Analysis
- SIP & AUM Analysis
- Portfolio Holdings Analysis

## Datasets
- Fund Master
- NAV History
- AUM Data
- SIP Inflows
- Category Inflows
- Industry Folio Count
- Scheme Performance
- Investor Transactions
- Portfolio Holdings
- Benchmark Indices
